Hotel Cancellation Prediction

Project Overview

This project aims to predict hotel booking cancellations using machine learning. The objective was to identify high-risk reservations and reduce revenue loss.

Business Problem

Cancellation rates increased from 15% to 37%, putting €4M annual revenue at risk. The goal was to build a predictive model to flag risky bookings.

Approach

  • Data cleaning and preprocessing
  • Exploratory Data Analysis (EDA)
  • Feature engineering
  • Model comparison (Logistic Regression, Decision Tree, Random Forest)
  • Evaluation using ROC-AUC & F1-score

Results

The best model achieved 82% accuracy and improved early detection of cancellation risk.
